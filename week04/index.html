Notes

  • Reset buttons can be considered bad practice
  • following sample code adds a submit listener:
    const form = document.forms['search']

    form.addEventListener('submit', search, false)

    function search() {
    // do stuff
    }
  • Use event.preventDefault() to prevent default submit behavior
  • input.value will give the current contents of the input (where input is a variable representing an input element)
  • Encapsulation - details kept from user
  • Polymorphism - some parts are reusable but still customizable
  • Inheritance - take features from another object and add some more
