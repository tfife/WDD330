Week 09 - Window and APIs

Team Activity

Notes

  • BOM = Browser Object model
  • window.alert('text'), window.prompt('text'), window.confirm('text')
  • Finding Browser Information
  • There's a navigator object. window.navitagor.userAgent will show the browser
  • window.location.href lets you change the location. You can also do window.location.assign()
  • Browser History
  • window.history will show previous pages visited before this page
  • window.history.go(-1) would go back one page. 0 would reload, 1 would go forward one
  • window.history.forward() and window.history.back() go forward and back in history
  • Controlling windows
  • window.screen.height will give the height as a number. window.screen.availHeight takes away operating system menus
  • Cookies
  • create cookie by saying document.cookie = 'variableName=value'
  • document.cookie will give you all current values in cookie jar.
  • use split to access individual cookies.
  • cookies are session cookies. To make them persistent use "; expires=date" when it's created
  • Timing functions
  • window.setTimeout(() => { //do stuff }, 3000) will set a 3000 timer and at the end will do the stuff. It will return an ID of which timeout it was
  • window.clearTimeout(5) when the id is 5
  • setInterval acts like setTimeout but will repeat

  • Make your own attributes. Name them data-whateverNameYouWant
  • localStorage
  • navigator.geolocation.getCurrentPosition(youAreHere) where youAreHere is a function which will automatically have a parameter giving a position. Position can be accessed with position.coords.latitude etc.
  • also use position.speed, .altitude .heading (direction) .timestamp
  • navigator.geolocation.watchPosition(youAreHere) is the same but updates with every change in position
  • Workers (not super sure what they do)
