Notes

  • use let instead of var because let has block scope and doesn't allow other bad stuff var can do. var may be seen in older code
  • use either camelCased or underscore_separated variable names
  • In the following code, changing d.value also changes c.value because d and c reference the same object
    const c = { value: 1 };
    let d = c; // c.value = 1, d.value = 1
    d.value = 2; // c.value = 2, d.value = 2
  • String functions
    • .toUpperCase() : string
    • .toLowerCase() : string
    • .charAt(index) : char
    • .indexOf(character) : number
    • .lastIndexOf(character) : number
    • .startsWith(character) : bool
    • .endsWith(character) : bool
    • .concat('string', 'anotherString') : string
  • use backslash to escape characters (quotes, newlines, tabs, \r).
  • Question: what is \r and how is it different from \n?
